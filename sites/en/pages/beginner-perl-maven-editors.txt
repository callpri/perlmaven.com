Before you get started programming in Perl, you need to pick an editor or and IDE to use. There are plenty of options.
Here are a few recommended ones.

=abstract end

<slidecast file="beginner-perl/editors" youtube="7mvy8sUcigM" />


<h2>Mostly for Linux/Unix</h2>
<ul>
  <li><a href="http://www.gnu.org/software/emacs/">Emacs</a></li>
  <li><a href="http://www.vim.org/">vi, vim, gvim </a></li>
  <li><a href="http://kate-editor.org/">Kate</a></li>
  <li><a href="http://projects.gnome.org/gedit/">Gedit</a></li>
</ul>

<h2>Programmer's editors for Windows</h2>
<ul>
  <li><a href="http://notepad-plus-plus.org/">Notepad++</a></li>
  <li><a href="http://www.textpad.com/">Textpad</a></li>
  <li><a href="http://www.ultraedit.com/">Ultra Edit</a></li>
  <li><a href="https://codelobster.com/">Codelobster</a></li>
</ul>

<h2>Editor for Mac</h2>
<ul>
  <li><a href="http://macromates.com/">TextMate</a></li>
</ul>

<h2>Platform independent</h2>
<ul>
  <li><a href="http://www.sublimetext.com/">Sublime Text</a></li>
</ul>

<h2>IDEs</h2>
<ul>
  <li><a href="http://www.activestate.com/">Komodo of ActiveState</a></li>
  <li><a href="http://www.eclipse.org/">Eclipse</a> with EPIC</li>
  <li>JetBrains has Perl support as well</li>
  <li>Visual Studio Code</li>
  <li>Atim</li>
</ul>

See also the article about <a href="/perl-editor">editors and IDEs</a> with some extra comments
and check out the <a href="http://perlide.org/poll200910/">Perl IDE and Editor Poll, October 2009</a> if you like that kind of stuff.

(Unfortunately since the recording of this video Padre is not developed any more. Its source code was moved to GitHub,
so it will be very easy for you to pick up the pieces and improve it yourself.)
